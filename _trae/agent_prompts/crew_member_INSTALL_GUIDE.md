# 执行组员 — TRAE 创建指南

## 表单字段速查

以下是在 TRAE GUI 中创建「执行组员」智能体时需要填写的所有字段值。

### 名称

```
执行组员
```

### 提示词

复制 `crew_member.md` 中 **`## Agent Configuration`** 及以下的全部内容，粘贴到提示词输入框。

> 提示词文件路径：`cursor-agent-team/_trae/agent_prompts/crew_member.md`

### 英文标识名

```
crew-member
```

### 何时调用

```
当需要严格按照 ai_workspace/plans/ 中的执行方案逐步执行操作时调用。包括：代码修改、文件创建、脚本执行、git操作、依赖安装等需要精确执行的开发任务。此智能体不做讨论，只按方案执行。
```

### 工具

- ✅ 文件系统
- ✅ 终端
- ✅ 联网搜索

---

## 创建步骤

1. 打开 TRAE → 右上角齿轮图标（设置）→ **智能体** 页签 → 点击 **+ 创建智能体**
2. 在「名称」字段输入：`执行组员`
3. 在「提示词」字段粘贴 `crew_member.md` 中 `## Agent Configuration` 及以下的全部内容
4. 在「英文标识名」字段输入：`crew-member`
5. 在「何时调用」字段粘贴上方「何时调用」区域的完整文本
6. 在「工具」区域勾选：文件系统、终端、联网搜索
7. 点击 **创建**

---

## 使用方式

创建完成后，在 TRAE 对话输入框中输入 `@执行组员` 即可调用。
