# cursor-agent-team — TRAE_CN Adaptation

TRAE_CN (字节跳动) 平台适配版本。将 cursor-agent-team 的单人多角色 AI 协作方法论从 Cursor IDE 迁移到 TRAE_CN。

## Prerequisites

- TRAE_CN v3.3.21+ (或 v1.3.0+)
- Python 3.8+
- Git

## Installation

### 1. Clone with submodule

Use the default branch (main); the TRAE adaptation is included.

```bash
git clone --recurse-submodules <your-project-repo>
cd <your-project>
```

### 2. Run installation script

```bash
bash cursor-agent-team/install_trae.sh
```

This will:
- Create `.trae/rules/` and `.trae/skills/` in your project root
- Copy `project_rules.md` to `.trae/rules/`
- Copy all 6 Skills to `.trae/skills/`

### 3. Create Agents in TRAE GUI (manual)

Create the three agents (讨论搭档, 执行组员, 提示工程师) in TRAE → Settings → Agent. For each agent, use the **per-agent install guide** so all form fields (名称, 提示词, 英文标识名, 何时调用, 工具) are filled correctly.

- Index and links: see `_trae/agent_prompts/README.md`
- Step-by-step with pre-filled values: see `_trae/agent_prompts/<name>_INSTALL_GUIDE.md` for 讨论搭档, 执行组员, 提示工程师

Note: 英文标识名 and 何时调用 are required in the TRAE GUI.

## Usage

### Discussion Partner (讨论搭档)

Used for discussion, analysis, brainstorming, and plan generation. Does NOT execute operations.

```
@Discussion Partner 我们来讨论一下项目架构
@Discussion Partner 分析一下这个方案的优劣
@Discussion Partner 生成执行方案
```

### Crew Member (执行组员)

Used for executing plans generated by Discussion Partner.

```
@Crew Member PLAN-AC-001
@Crew Member 执行最新的方案
```

### TRAE Prompt Engineer (TRAE 提示工程师)

Used for creating and maintaining Agent Prompts, Skills, and Rules.

```
@TRAE Prompt Engineer 创建一个新的写作助手 Agent
@TRAE Prompt Engineer 更新 Discussion Partner 的提示词
```

## Differences from Cursor Version

| Aspect | Cursor | TRAE_CN |
|---|---|---|
| Role activation | `/discuss`, `/crew`, `/prompt_engineer` commands | `@Discussion Partner`, `@Crew Member`, `@TRAE Prompt Engineer` agents |
| Rules injection | `.mdc` files auto-injected by Cursor | Project Rules in `.trae/rules/`, Skills in `.trae/skills/` |
| Cross-cutting aspects | Always-apply `.mdc` rules | Skills loaded by TRAE based on relevance |
| Agent creation | Defined in `.cursor/commands/` | Manual GUI creation + prompt copy-paste |
| Scripts | Identical | Identical |
| AI workspace | Identical | Identical |

## Updating Agent Prompts

When Agent Prompts are updated in `_trae/agent_prompts/`:
1. Pull latest changes: `cd cursor-agent-team && git pull`
2. Open TRAE → Settings → Agent
3. Find the Agent to update
4. Replace the prompt text with the new content from the updated file
5. Save

## Updating Skills and Rules

```bash
bash cursor-agent-team/install_trae.sh
```

This re-copies rules and skills to `.trae/`.
