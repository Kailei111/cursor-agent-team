# Agent Requirements

Storage for agent requirements generated by `/discuss`. Path: `cursor-agent-team/ai_workspace/agent_requirements/`

---

## Quick Reference

| What | Path | Format |
|------|------|--------|
| Requirement File | `agent_requirements/` | `AGENT-REQUIREMENT-[TopicID]-[Seq].md` |
| Index | `agent_requirements/` | `INDEX.md` |

Examples:
- `AGENT-REQUIREMENT-A-001.md` - Topic A, Requirement 1
- `AGENT-REQUIREMENT-B-002.md` - Topic B, Requirement 2

---

## NEVER DO

- NEVER delete `INDEX.md`
- NEVER delete this README
- NEVER create requirements outside this directory

---

## Workflow

1. `/discuss` generates requirement → saves to `AGENT-REQUIREMENT-[TopicID]-[Seq].md`
2. `/discuss` updates `INDEX.md`
3. `/prompt_engineer` processes requirement
4. Processor updates requirement status and processing records
